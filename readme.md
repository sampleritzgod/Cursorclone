# Cursor Landing Page Clone

A modern responsive landing page inspired by Cursor AI website.

This project is built using pure HTML and CSS only.
It demonstrates layout design, responsive UI, flexbox, and clean styling.

---

## Features

- Fixed header navigation bar
- Hero section with call-to-action button
- Responsive design (mobile + desktop)
- Modern dark theme UI
- Flexbox layout
- Pure HTML + CSS

---

## Technologies Used

- HTML5
- CSS3
- Flexbox
- Media Queries (Responsive Design)

---

## How to Run

1. Download or clone the project
2. Open `Cursor.html` in any browser

OR

Double-click the file to open directly.

No installation required.

---

## What I Learned

- Creating responsive layouts
- Using Flexbox for alignment
- Header positioning (fixed navbar)
- CSS variables
- Clean UI design principles
